## Overview

<a href="#">Archiver 4</a> brings a clean and capable archive management experience to Mac, handling compression and extraction tasks that the built-in tools cannot. The application supports an extensive list of formats far beyond what macOS handles natively, making it the reliable choice whenever an unusual archive format arrives or a specific output format is required for compatibility.

The <a href="#">drag-and-drop interface</a> keeps operations simple — drop files onto the window to compress, drop archives to extract. No menu navigation or format selection dialogs interrupt the workflow for standard operations. <a href="#">Format detection</a> happens automatically when archives are dropped for extraction, identifying the format and applying the correct decompression method without user input. The interface stays clean and focused regardless of what you throw at it.

<a href="#">Password protection</a> encrypts archive contents with AES encryption for sensitive files. Set passwords during creation to lock archives before sharing. Archiver prompts for passwords automatically when opening protected files. The <a href="#">Secret Safe feature</a> creates encrypted vaults that store sensitive documents securely, functioning as a simple encrypted container for files you want to protect but not necessarily share.

The <a href="#">preview system</a> lets you inspect archive contents before extracting everything. Browse file trees inside archives, open individual files without full extraction, and selectively pull out only the items you need. <a href="#">Splitting large archives</a> divides oversized files into smaller chunks for easier transfer across size-limited channels. These chunks reassemble automatically during extraction into the complete original content.

---

## Key Features

- <a href="#">Multi-format support</a> handles ZIP RAR 7-ZIP TAR GZ and dozens of other archive formats natively
- Simple <a href="#">drag-and-drop workflow</a> compresses or extracts files without navigating any menus or dialogs
- <a href="#">AES encryption</a> protects archive contents with strong password-based security before sharing
- Built-in <a href="#">Secret Safe</a> creates encrypted vaults for storing sensitive documents securely on your Mac
- <a href="#">Archive preview</a> lets you browse and open individual files without extracting the entire contents first
- <a href="#">Selective extraction</a> pulls out only the specific files you need from any archive without full unpacking
- <a href="#">Archive splitting</a> divides oversized files into smaller chunks that reassemble automatically on extraction
- <a href="#">Compression presets</a> let you choose between fastest speed and maximum size reduction as needed
- <a href="#">Finder integration</a> adds compress and extract actions directly to the right-click context menu
